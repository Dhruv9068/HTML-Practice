HTML Forms

Forms are used to collect user input. They contain various input elements like text fields, checkboxes, radio buttons, and more.

Complete Form Example

Common Input Types:
  • text - Single-line text input
  • email - Email address input (with validation)
  • password - Password input (masked text)
  • number - Numeric input
  • date - Date picker
  • tel - Telephone number
  • radio - Radio button (single selection)
  • checkbox - Checkbox (multiple selections)
  • file - File upload
  • submit - Submit button
  • reset - Reset button

Other Input Types (HTML5)

Form Attributes:
  • action - URL where form data is sent
  • method - HTTP method (GET or POST)
  • required - Makes a field mandatory
  • placeholder - Hint text in input field
  • disabled - Disables the input
  • readonly - Makes input read-only
  • min/max - Min/max values for number inputs
  • pattern - Regular expression for validation
