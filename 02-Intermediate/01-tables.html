HTML Tables

Tables are used to organize data in rows and columns. They're perfect for displaying structured information.

Table Elements:
  • <table> - Defines the table
  • <thead> - Groups header content
  • <tbody> - Groups body content
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell
  • <caption> - Table caption

Spanning Attributes:
  • colspan - Spans across multiple columns
  • rowspan - Spans across multiple rows

Best Practices:
  • Use tables for tabular data, not for layout
  • Always include a caption for accessibility
  • Use <thead>, <tbody>, and <tfoot> to organize content
  • Use <th> for header cells instead of <td>
