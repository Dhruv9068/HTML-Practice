HTML5 Multimedia

HTML5 introduced native support for audio and video elements, making it easy to embed multimedia content without plugins.

Video Element

The <video> tag embeds video content directly into your web page.

Sample video with controls

Video Code Example:

<video width="640" height="360" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.ogg" type="video/ogg">
    Your browser does not support the video tag.
</video>
Video Attributes:
  • controls - Shows playback controls
  • autoplay - Starts playing automatically
  • loop - Loops the video
  • muted - Mutes the audio
  • poster - Image shown before video plays
  • width/height - Dimensions of the video
  • preload - How the video should be loaded (auto, metadata, none)

Audio Element

The <audio> tag embeds audio content into your web page.

Sample audio with controls

Audio Code Example:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>
Audio Attributes:
  • controls - Shows playback controls
  • autoplay - Starts playing automatically
  • loop - Loops the audio
  • muted - Mutes the audio
  • preload - How the audio should be loaded

Embedding YouTube Videos

You can embed YouTube videos using an <iframe> tag:

Embedded YouTube video

YouTube Embed Code:

<iframe width="640" height="360"
    src="https://www.youtube.com/embed/VIDEO_ID"
    allowfullscreen>
</iframe>

Supported Media Formats

Video Formats:

  • MP4 - Widely supported, H.264 codec
  • WebM - Open format, VP8/VP9 codec
  • Ogg - Open format, Theora codec

Audio Formats:

  • MP3 - Most widely supported
  • WAV - Uncompressed audio
  • Ogg - Open format, Vorbis codec
  • AAC - Advanced Audio Coding
Best Practices:
  • Always provide multiple source formats for better browser compatibility
  • Include fallback text for browsers that don't support media elements
  • Use the controls attribute to let users control playback
  • Be cautious with autoplay - many browsers block it
  • Optimize media files for web to reduce file size and improve loading
  • Always include captions/subtitles for accessibility

Advanced: Video with Subtitles

<video width="640" height="360" controls>
    <source src="video.mp4" type="video/mp4">
    <track kind="subtitles" src="subtitles_en.vtt" 
           srclang="en" label="English">
    <track kind="subtitles" src="subtitles_es.vtt" 
           srclang="es" label="Spanish">
</video>

The <track> element adds text tracks (subtitles, captions) to media elements.
