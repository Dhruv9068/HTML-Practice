Text Formatting in HTML

HTML provides various tags to format text and give it different meanings.

Common Text Formatting Tags

Bold text - Created with <strong> (semantic) or <b> (visual only)

Italic text - Created with <em> (semantic) or <i> (visual only)

Underlined text - Created with <u>

Highlighted text - Created with <mark>

Smaller text - Created with <small>

Deleted text - Created with <del>

Inserted text - Created with <ins>

This is subscript text - Created with <sub>

This is superscript text - Created with <sup>

Code and Preformatted Text

Use <code> for inline code: let x = 10;

Use <pre> for preformatted text that preserves spaces and line breaks:

function greet() {
    console.log("Hello World!");
    return true;
}

Quotations

Short inline quote: To be or not to be

This is a longer quotation that appears in a block. It's created with the <blockquote> tag and is typically indented by browsers.

Citation example: The Great Gatsby by F. Scott Fitzgerald

Text Formatting Summary Table

Tag Purpose Example

<del> Deleted text deleted

Semantic vs Visual:

Use semantic tags like <strong> and <em> instead of <b> and <i> when possible. Semantic tags convey meaning to search engines and screen readers, not just visual styling.
