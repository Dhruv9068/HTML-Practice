Basic HTML Structure

Every HTML document follows a basic structure. Let's understand each part:

HTML Document Structure:

  • <!DOCTYPE html> - Declares this is an HTML5 document
  • <html> - The root element of an HTML page
  • <head> - Contains meta information about the document
  • <title> - Specifies the title shown in the browser tab
  • <body> - Contains the visible page content

Essential Head Elements:

  • <meta charset="UTF-8"> - Sets character encoding to support all languages
  • <meta name="viewport"> - Makes the page responsive on mobile devices
  • <title> - Page title that appears in browser tab

Why This Structure Matters:

This basic structure is the foundation of every web page. The DOCTYPE tells browsers to use modern standards, the head contains metadata, and the body contains everything users see.
