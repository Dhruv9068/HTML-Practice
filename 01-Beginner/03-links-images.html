Links and Images

HTML Links

Links allow you to navigate between pages. They are created using the <a> (anchor) tag.

Link Examples:

External Link:

Visit W3Schools (opens in new tab)

Internal Link:

Go to Home Page

Email Link:

Send Email

Phone Link:

Call Us
Link Attributes:
  • href - The destination URL
  • target="_blank" - Opens link in new tab
  • title - Tooltip text on hover

HTML Images

Images make your webpage more engaging. Use the <img> tag to add images.

Image Example:

This is a placeholder image example

Image Attributes:
  • src - Path to the image file (required)
  • alt - Alternative text for accessibility (required)
  • width - Image width in pixels
  • height - Image height in pixels
  • title - Tooltip text on hover

Image as a Link

You can make an image clickable by wrapping it in an anchor tag:

Click the image above!

Best Practice: Always include the alt attribute for images. It helps with accessibility and SEO, and shows text if the image fails to load.
