HTML Meta Tags and SEO

Meta tags provide metadata about your HTML document. They are placed in the <head> section and are not visible on the page but are crucial for SEO, social media sharing, and browser behavior.

Essential Meta Tags

1. Character Encoding

<meta charset="UTF-8">

Specifies the character encoding for the HTML document. UTF-8 supports all languages.

2. Viewport (Mobile Responsive)

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Essential for responsive design. Controls the viewport width and zoom level on mobile devices.

3. IE Compatibility

<meta http-equiv="X-UA-Compatible" content="IE=edge">

Ensures Internet Explorer uses the latest rendering engine.

SEO Meta Tags

Title Tag

<title>Page Title - 50-60 Characters Ideal</title>

The most important SEO element. Appears in search results and browser tabs.

Description

<meta name="description" content="A brief description of the page content. Ideal length is 150-160 characters.">

Appears in search results below the title. Should be compelling and include keywords.

Keywords (Less Important Now)

<meta name="keywords" content="keyword1, keyword2, keyword3">

Once important for SEO, now mostly ignored by search engines.

Robots

<meta name="robots" content="index, follow">

Controls how search engines crawl and index your page.

  • index - Allow search engines to index this page
  • noindex - Prevent indexing
  • follow - Follow links on this page
  • nofollow - Don't follow links

Author

<meta name="author" content="Author Name">

Specifies the author of the document.

Open Graph Meta Tags (Facebook)

Open Graph meta tags control how your content appears when shared on Facebook and other social platforms.

<meta property="og:type" content="website">
<meta property="og:url" content="https://example.com/page">
<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page description">
<meta property="og:image" content="https://example.com/image.jpg">

Twitter Card Meta Tags

Twitter Cards allow you to attach rich photos, videos, and media to Tweets.

<meta property="twitter:card" content="summary_large_image">
<meta property="twitter:url" content="https://example.com/page">
<meta property="twitter:title" content="Page Title">
<meta property="twitter:description" content="Page description">
<meta property="twitter:image" content="https://example.com/image.jpg">

Twitter Card Types:

  • summary - Standard card with small image
  • summary_large_image - Large image card
  • app - Mobile app download card
  • player - Video/audio player card

Other Useful Meta Tags

Meta Tag Purpose Example
Refresh Auto-refresh page <meta http-equiv="refresh" content="30">
Theme Color Browser toolbar color <meta name="theme-color" content="#667eea">
Language Page language <html lang="en">
Canonical Preferred URL <link rel="canonical" href="https://example.com/page">

Complete Example

Here's a complete <head> section with all important meta tags:

<head>
    <!-- Essential -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- SEO -->
    <title>Your Page Title</title>
    <meta name="description" content="Your page description">
    <meta name="keywords" content="keyword1, keyword2">
    <meta name="author" content="Your Name">
    <meta name="robots" content="index, follow">
    
    <!-- Open Graph / Facebook -->
    <meta property="og:type" content="website">
    <meta property="og:url" content="https://example.com">
    <meta property="og:title" content="Your Page Title">
    <meta property="og:description" content="Your page description">
    <meta property="og:image" content="image-url.jpg">
    
    <!-- Twitter -->
    <meta property="twitter:card" content="summary_large_image">
    <meta property="twitter:url" content="https://example.com">
    <meta property="twitter:title" content="Your Page Title">
    <meta property="twitter:description" content="Your page description">
    <meta property="twitter:image" content="image-url.jpg">
    
    <!-- Favicon -->
    <link rel="icon" type="image/png" href="favicon.png">
</head>
Best Practices:
  • Keep title tags under 60 characters
  • Keep descriptions between 150-160 characters
  • Use unique meta tags for each page
  • Include target keywords naturally in title and description
  • Use high-quality images (1200x630px) for social media previews
  • Always test your social media cards before publishing
⚠️ Common Mistakes:
  • Duplicate title and description tags across pages
  • Using the same image for all pages' Open Graph tags
  • Keyword stuffing in meta keywords
  • Missing viewport meta tag (breaks mobile experience)
  • Title tags that are too long (truncated in search results)
